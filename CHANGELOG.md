## [0.1.0] - 2026-03-04

### Added

- Next.js 16 app router
- User authentication — register, login, logout pages
- Protected routes with middleware
- Dashboard — project stats and event counts
- Projects list with API key management
- Project detail with live event feed via Socket.IO
- Copy API key button
- Real-time events stream without page refresh
- Vercel deployment configuration
